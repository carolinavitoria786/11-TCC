# 07-jcup

(muda repositorio no terminal)
cd TCC-Parte2-Jflex-Jcup/Praticas-Jcup/Pratica-11/Questão-2-JSON

1. Dar permissão de execução para o arquivo de script `executar.sh` (torná-lo executável):

- `./executar.sh`
